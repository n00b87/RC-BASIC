COMMENTS
    Comments in RC BASIC have to be on a separate line from executable code.  To make a comment you would use a single quote.  Look at the following:

    ' THIS IS A CORRECT COMMENT
    Print "HELLO WORLD"      ' THIS IS AN INCORRECT COMMENT.  COMMENTS MUST BE ON SEPARATE LINES FROM CODE

    In the above example the first line is correct.  It starts the line with a comment.  The second line however attempts to add a comment to a line that already has code on it.  That line will not compile.
